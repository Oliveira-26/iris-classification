# Classificação de Flores Iris com Machine Learning

Meu primeiro projeto de ML. Usei o dataset Iris do scikit-learn pra treinar um modelo de Decision Tree que classifica flores em 3 espécies com base nas medidas de pétalas e sépalas.

## Resultado

- **Modelo:** Decision Tree
- **Acurácia:** 100%
